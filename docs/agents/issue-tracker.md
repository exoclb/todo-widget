# Issue Tracker

Issues for this repo are tracked in GitHub Issues.

Repository:

```txt
exoclb/todo-widget
```

Publishing workflow:

- Create issues in GitHub Issues for `exoclb/todo-widget`.
- Publish issues in dependency order when one issue blocks another.
- Use the repo triage labels documented in `docs/agents/triage-labels.md`.
- Keep issue descriptions aligned with `CONTEXT.md` and the ADRs in `docs/adr/`.
