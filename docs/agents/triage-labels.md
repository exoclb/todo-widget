# Triage Labels

This repo uses the default triage label vocabulary.

```txt
needs-triage
needs-info
ready-for-agent
ready-for-human
wontfix
```

Canonical roles:

- `needs-triage`: maintainer needs to evaluate the issue.
- `needs-info`: waiting on the reporter for more information.
- `ready-for-agent`: fully specified and ready for an AFK agent.
- `ready-for-human`: ready for a human implementer.
- `wontfix`: will not be actioned.
